# Curtaincall
### 기상 API와 온도, 조도 센서를 이용해 사용자 맞춤에 따른 4가지 모드를 지원하는 스마트 블라인드 앱


* 일반 모드
  > 앱 내부의 커튼 조절키를 사용해 높이를 조절
  
* 에너지 효율 모드
  > 적정 온도에 따라 냉방 및 난방 비용을 절약하기 위해 커튼을 자동 조절
  
* 조경 모드
  > 조도에 따라 알맞은 조경 환경을 조성
  
* 방범 모드
  > 사용자가 설정한 시간 간격마다 커튼을 개폐
  
* 알람 모드
  > 사용자가 설정한 시간에 커튼을 개폐
